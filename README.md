## 简介  
主要运用技术：  
1. python3作为平台开发和实现语言
2. Pycharm为开发IDE
3. python相关爬虫库
4. 数据存储与整理：EXCEL与MySQL数据库
5. 运用Flask框架进行web前端开发
6. Echarts实现数据可视化的展示

主要工作内容为：  
1. 数据的选择  
2. 数据的爬取
3. 数据的存储与清洗
4. web平台的设计与实现
